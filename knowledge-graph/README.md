# Knowledge Graph System

Pre-generated AI-powered template connections for the Life OS system.

## Quick Start

### 1. Generate Static Knowledge Graph
```bash
# Set your Claude API key
export ANTHROPIC_API_KEY="your_key_here"

# Generate the static graph (run once)
cd knowledge-graph
node generate-static-graph.js
```

### 2. Use in Your App
```javascript
const { knowledgeGraph } = require('./knowledge-graph/use-static-graph');

// Get related templates
const related = knowledgeGraph.getRelatedTemplates('home-buying');

// Get cross-template prompts
const prompts = knowledgeGraph.getCrossTemplatePrompts(['home-buying', 'wedding-planning']);
```

## Files

### Core System
- `generate-static-graph.js` - Generates all connections using Claude AI
- `use-static-graph.js` - Use pre-generated connections in your app

### Generated Data (created after running generator)
- `static-knowledge-graph.json` - Complete generated dataset
- `template-connections.json` - Just the relationships
- `cross-template-prompts.json` - Just the prompts
- `life-os-config.json` - Life OS integration settings

### Legacy Analysis
- `real-analysis.js` - Template category analysis
- `template-families.js` - Manual template families
- `ai-enhanced-connections.js` - Semantic similarity scoring

## API

### Get Related Templates
```javascript
const related = knowledgeGraph.getRelatedTemplates('template-id', limit);
// Returns: [{ templateId, relationship, strength, reason }]
```

### Get Cross-Template Prompts
```javascript
const prompts = knowledgeGraph.getCrossTemplatePrompts(['template1', 'template2']);
// Returns: [{ prompt, type, source, applicableTemplates, priority }]
```

### Get Influenced Templates
```javascript
const influenced = knowledgeGraph.getInfluencedTemplates('divorce-coordination');
// Returns: [{ templateId, impact, reason }]
```

## Connection Types

- **Semantic Clusters**: Templates grouped by life themes (e.g., "young_adult_foundation")
- **Cross-Connections**: Templates naturally used together (e.g., home-buying ↔ budget-planning)
- **Life Sequences**: Decision progression paths (e.g., college → job-search → apartment-hunting)
- **Influence Relationships**: How decisions affect other areas (e.g., divorce impacts financial planning)

## Why Static Generation?

- **Performance**: No API calls during app usage
- **Cost**: Generate once, use forever
- **Reliability**: No dependency on external APIs in production
- **Intelligence**: Still uses Claude AI, just pre-computed

All connections are generated by Claude AI based on psychological decision-making patterns, not keyword matching.
