# Screenshots Needed for Landing Page

## Quick Guide
1. Open the app in your browser
2. Use browser DevTools (F12) → Device Toolbar (Cmd+Shift+M) → Set to Desktop HD (1920x1080)
3. Take screenshots using Cmd+Shift+4 (Mac) or Snipping Tool (Windows)
4. Save them with these exact names in this folder

---

## Pro Tips
- Use dark mode or light mode consistently
- Clear any personal data first
- Make sure UI looks clean (no weird scrollbars)
- High resolution is better (we'll optimize later)
- Use actual content, not lorem ipsum
